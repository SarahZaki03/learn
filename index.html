Terminal

PowerShell: The terminal is a program that allows you to interact with your computer by entering commands. You already have a program called Windows PowerShell.

Important commands:

  • cd <directory> The cd (change directory) command shows or changes the current location.
  • The directory parameter is optional, and without it, the command prints the current working directory.
  • cd .. The current directory changes to one directory above the current location.
  • cls Command: The cls command clears the text in a command prompt window and returns a blank surface. Use the command to clear the screen contents.
  • md <directory> The md and mkdir (make directory) commands create a new directory or subdirectory.
  • ls lists files and directories. If the pathname is a file, ls displays information on the file according to the requested options. If the pathname is a directory, ls displays information on the files and subdirectories therein.
